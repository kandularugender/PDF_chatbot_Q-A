# PDF Chatbot Q&A

This project is a Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions from them.

## Features
- Upload PDF files
- Extract text from documents
- Store embeddings in a vector database
- Ask questions and get AI-generated answers

## Tech Stack
- Python
- Streamlit
- LangChain
- ChromaDB
- Ollama (Mistral model)

## Run the Project

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
